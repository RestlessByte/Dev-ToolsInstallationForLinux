# Tools for Linux Installation

## Installation

```bash
git clone git@github.com:RestlessByte/RestlessByte-Dev-ToolsInstallationForLinux.git
```

## Usage

```bash
git clone git@github.com:RestlessByte/RestlessByte-Dev-ToolsInstallationForLinux.git
alias installTools='bash /home/$USER/toolsForLinuxInstallation/menu.sh'
source ~/.bashrc
installTools
```
